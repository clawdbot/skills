# Gettr Transcript & Summary

Download a video from a GETTR post, transcribe it locally with MLX Whisper, and produce a clean summary or timestamped outline.

## What it does
- Extracts the MP4 from a GETTR post (via og:video)
- Transcribes locally (no API keys required)
- Summarizes into bullets or a timestamped outline

## Example
Provide a GETTR post URL and ask for a transcript or summary; the skill will download, transcribe, and summarize.
